Given an array arr[] of length n. Find all possible unique permutations of the array.


Example 1:

        Input: 
        n = 3
        arr[] = {1, 2, 1}
        Output: 
        1 1 2
        1 2 1
        2 1 1
Explanation:
These are the only possible unique permutations
for the given array.
Example 2:

        Input: 
        n = 2
        arr[] = {4, 5}
        Output: 
        4 5
        5 4
